# SINODREAM - 专业中国移民咨询

欢迎访问 SINODREAM 移民咨询机构官方网站

## 关于我们

SINODREAM 是一家专业的移民咨询机构，致力于帮助海外人士实现移民中国的梦想。我们提供全方位的移民咨询服务，从签证申请到居留许可，让您的移民之路畅通无阻。

## 网站功能

本网站提供以下功能：

- **服务介绍** - 展示6大移民服务（工作签证、永久居留、家庭团聚、学习签证、商务签证、居留延期）
- **关于我们** - 展示SINODREAM的4大优势
- **办理流程** - 详细的5步移民办理流程
- **在线咨询** - 客户可以通过表单提交咨询请求
- **联系方式** - 提供邮箱、电话、微信等多种联系方式

## 如何使用

1. 在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   python3 -m http.server 8000
   ```
   然后访问 `http://localhost:8000`

## 文件结构

- `index.html` - 主页面
- `styles.css` - 样式表
- `script.js` - JavaScript交互脚本
